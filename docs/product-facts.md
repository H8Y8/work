# Claude-inspired visual reference facts

> Verified: 2026-07-22

- The current official Anthropic site uses a warm ivory base (`#faf9f5`), dark slate text (`#141413`), clay accent (`#d97757`) and a softer ivory surface (`#f0eee6`).
- Its typography system pairs a display serif with a sans-serif detail face, while paragraph text is also serif-led.
- The official design uses generous section spacing, restrained borders, high-contrast primary buttons and limited accent color.
- This project borrows those editorial principles only. It does not use Anthropic or Claude logos, proprietary fonts, illustrations, or claim affiliation.
