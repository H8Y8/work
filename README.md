# 凌華科技 ADLINK｜IE 面試作戰手冊

手機優先的一頁式面試準備網站，整理凌華科技的公司定位、產品線、製造流程、IE 方法、成本模型、KPI、預算與面試問題。

## 網站

<https://h8y8.github.io/work/>

## 功能

- 10 分鐘面試衝刺模式
- 手機底部章節導覽與閱讀進度
- 可點擊、搜尋的專業名詞庫
- Layout、成本、KPI 與預算回答框架
- 三個必備案例與反問題清單
- 官方資料來源與查證範圍說明

## 本機開發

```bash
npm install
npm run dev
```

## 驗證

```bash
npm run lint
npm test
npm run pages:build
```

推送到 `main` 後，GitHub Actions 會自動建置並部署 GitHub Pages。
